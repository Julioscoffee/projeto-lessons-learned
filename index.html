Dinâmica do Git

Um dos dias que mais gostei até aqui foi o dia que tivemos uma dinâmica de grupo para entender porque foi criado e qual a importância do Git na vida de um desenvolvedor.

Principais comandos do Git

Comando Descrição
git clone ssh-do-repositório Usado para clonar o repositório do GitHub sd-013-a-project-lessons-learned
git status Verifica o que está fora do stage (em vermelho) e o que está dentro do stage (em verde)
git add Adiciona arquivos ao stage do Git
git commit Cria um commit com os arquivos que estão no stage do Git
git push -u nome-da-branch Envia o commit para o repositório remoto e torna a branch como padrão para os próximos push`s

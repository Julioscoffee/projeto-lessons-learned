Os meus primeiros passos na Trybe

Novos passos

Seria um clichê mencionar que nem toda a mudança é harmoniosa. Mas, dentro das possibilidades caóticas oriundas de um processo de transformação, mesmo a bagunça pode ser ordenada em oportunidades para desenvolvimento.

Quando decidi mudar de carreira, migrando dos estudos na área das ciências sociais aplicadas para o desafiador campo da tecnologia, encontrei na Trybe um caminho para crescer de forma ágil e constante. Ao longo dos primeiros dias de estudo, esta página é uma simbólica demonstração de que, passo a passo, a Trybe nos impulsiona a decolar mesmo em meio aos desafios cotidianos.

Dentro dessa jornada incrivelmente desafiadora, nossa formação na Trybe nos ensina, logo nas duas primeiras semanas, as seguintes competências:

  1. Fundamentos do Desenvolvimento WEB (Unix e Bash)
  2. Git
  3. GitHub
  4. Princípios da Internet
  5. Estruturas de página
  6. Intro. HTML
  7. Seletores de posicionamento
  8. HTML Semântico
  9. Intro. CSS
  10. Intro. JavaScript

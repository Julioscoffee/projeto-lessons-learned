O que estou aprendendo?

HTML

HTML é a abreviação para a expressão inglesa HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto. O HTML define o que vai em cima, o que é texto, o que é coluna, o que é imagem. Ele estrutura muito bem a página com início, meio e fim. Ou, se preferir, head, body e footer (cabeça, corpo e pé).

CSS

CSS – Cascading Style Sheets – é um mecanismo para adicionar estilo como cores, fontes, espaçamento, entre outras a um documento web. Imagine aquele cara que é bem mais ou menos, só que ao vestir uma roupa da hora fica super cool. Ele dá atratividade à sua criação web.

Comandos git no automático

Seria injusto não citar a grande dupla GIT e GitHub, presentes em praticamente todos os momentos de estudo e de projeto. O passo a passo básico já está automático. Um pequeno resumo dos principais comandos logo abaixo:

Comando Função
git status printa estado atual dos arquivos
git add . adiciona todo conteudo alterado ao staging
git commit -m adiciona conteúdos do staging ao repositório local
git push envia o conteúdo do commit para o repositório remoto

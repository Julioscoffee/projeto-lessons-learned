O que estou aprendendo?

HTML é a abreviação para a expressão inglesa HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto. O HTML define o que vai em cima, o que é texto, o que é coluna, o que é imagem. Ele estrutura muito a página.
CSS – Cascading Style Sheets – é um mecanismo para adicionar estilo como cores, fontes, espaçamento, entre outras a um documento web. Ele dá atratividade à sua criação web.
